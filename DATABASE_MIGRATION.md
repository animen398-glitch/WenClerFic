# Миграция на базу данных SQLite

Все функции обновлены для использования SQLite вместо массивов в памяти.

## Что изменилось:

1. **Добавлен модуль `database.js`** - содержит все функции для работы с БД
2. **Создана папка `data/`** - здесь хранится файл базы данных `wenclerfic.db`
3. **Все данные теперь сохраняются** - при перезапуске сервера данные не теряются

## Установка:

```bash
npm install
```

## Запуск:

```bash
npm start
```

При первом запуске автоматически создастся база данных в папке `data/`.

## Структура базы данных:

- **users** - пользователи
- **fics** - фанфики
- **chapters** - главы
- **comments** - комментарии

## Важно:

- Файл базы данных `data/wenclerfic.db` добавлен в `.gitignore` - не коммитится в репозиторий
- Для продакшена рекомендуется использовать PostgreSQL или MongoDB
- Пароли пока хранятся в открытом виде - нужно добавить bcrypt
