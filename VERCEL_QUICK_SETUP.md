# Быстрая настройка Vercel

## Что нужно сделать в интерфейсе Vercel:

### ✅ Настройки сборки:

1. **Команда сборки (Build Command):**
   - Включите переключатель (сдвиньте вправо)
   - Оставьте пустым ИЛИ укажите: `npm install`

2. **Выходной каталог (Output Directory):**
   - Оставьте выключенным (как есть) ✅

3. **Установить команду (Install Command):**
   - **ВКЛЮЧИТЕ переключатель** (сдвиньте вправо) ⚠️ ВАЖНО!
   - Укажите: `npm install`

### ✅ Переменные среды:

**Удалите:** `EXAMPLE_NAME` (нажмите кнопку "-")

**Добавьте следующие переменные:**

1. Нажмите "+ Добавить еще"
   - Ключ: `GOOGLE_CLIENT_ID`
   - Значение: ваш Google Client ID

2. Нажмите "+ Добавить еще"
   - Ключ: `GOOGLE_CLIENT_SECRET`
   - Значение: ваш Google Client Secret

3. Нажмите "+ Добавить еще"
   - Ключ: `GOOGLE_REDIRECT_URI`
   - Значение: `https://ваш-проект.vercel.app/api/auth/google/callback`
   - ⚠️ Замените `ваш-проект` на реальное имя после первого деплоя

4. (Опционально) Если используете Facebook:
   - `FACEBOOK_APP_ID`
   - `FACEBOOK_APP_SECRET`
   - `FACEBOOK_REDIRECT_URI` = `https://ваш-проект.vercel.app/api/auth/facebook/callback`

### ✅ После настройки:

1. Нажмите **"Развертывать"** (Deploy)
2. Дождитесь завершения (1-2 минуты)
3. Скопируйте URL проекта (например: `https://wenclerfic-xxx.vercel.app`)
4. Обновите `GOOGLE_REDIRECT_URI` с правильным URL
5. Передеплойте проект

## ⚠️ Важно:

- Файл `vercel.json` уже настроен - Vercel автоматически определит конфигурацию
- База данных SQLite будет работать
- Все данные сохраняются в `data/wenclerfic.db`
