# Инструкция по запуску WenClerFic

## Быстрый старт

1. **Откройте терминал в папке `ficbook`**

2. **Установите зависимости:**
   ```bash
   npm install
   ```

3. **Запустите сервер:**
   ```bash
   npm start
   ```
   
   Или используйте готовый скрипт:
   ```bash
   start.bat
   ```

4. **Откройте браузер и перейдите на:**
   ```
   http://localhost:3000
   ```

## API Endpoints

### Аутентификация
- `POST /api/auth/register` - Регистрация
- `POST /api/auth/login` - Вход

### Фанфики
- `GET /api/fics` - Список фанфиков (query: page, genre, rating, sort)
- `GET /api/fics/:id` - Получить фанфик
- `POST /api/fics` - Создать фанфик

### Комментарии
- `GET /api/fics/:ficId/comments` - Список комментариев
- `POST /api/fics/:ficId/comments` - Создать комментарий

## Примечания

- Текущая версия использует in-memory хранилище (данные теряются при перезапуске)
- Пароли хранятся в открытом виде (для продакшена нужно использовать bcrypt)
- Токены простые (для продакшена использовать JWT)
- Нет валидации данных на сервере (нужно добавить)

## Проблемы?

Если что-то не работает:
1. Убедитесь, что порт 3000 свободен
2. Проверьте, что установлены все зависимости (`npm install`)
3. Проверьте консоль браузера на ошибки
4. Проверьте консоль сервера на ошибки
