# Быстрый старт WenClerFic

## 1. Установка зависимостей

```bash
cd C:\Users\Hiccup\Documents\LunarPortal\WenClerFic
npm install
```

## 2. Настройка OAuth (опционально)

Если хотите использовать OAuth авторизацию:

1. Следуйте инструкции в файле `КАК_ПОЛУЧИТЬ_OAUTH_КЛЮЧИ.md`
2. Создайте файл `.env` на основе `.env.example`
3. Добавьте свои OAuth ключи в `.env`

## 3. Запуск сервера

```bash
npm start
```

Или используйте готовый скрипт:
```bash
start.bat
```

## 4. Откройте в браузере

```
http://localhost:3000
```

## 5. Загрузка в GitHub

Следуйте инструкции в файле `КАК_ЗАГРУЗИТЬ_В_GITHUB.md`

## Тестовые аккаунты (для демо)

Если OAuth не настроен, можно использовать тестовые аккаунты:

- Email: `author1@test.com`, Пароль: `password123`
- Email: `author2@test.com`, Пароль: `password123`
- Email: `author3@test.com`, Пароль: `password123`

Эти аккаунты создаются автоматически при запуске сервера.
